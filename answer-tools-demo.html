Standalone module test

Answer Tools Demo

This page tests the new reusable Answer Tools module without touching app.html. The same module can later be attached to generated answers across Homework Helper, Explain Simply, Revision and Test Builder.

What this proves

The answer area, read aloud control, translation control, original/translated toggle and copy action can live outside the main app file. Once this works reliably, we can wire it into one existing tool at a time.
